Structure and Style

Follows separation of concerns principle. HTML5 defines the structure and content. !DOCTYPE informs browser to interpret document as HTML5. The html element is the root element. The head element contains metadata and links to external resources.

CSS3 defines the layout and style of HTML content. The stylesheet is linked in the head element using the link element. It enables consistent and reusable styling across multiple pages. The display property in CSS is crucial for controlling how elements are rendered, defining the type of box an element generates, impacting layout and visibility.

In this project I used tag to targets all instances of an HTML tag, class selectors to apply style to the attribute class, and inline styling.
